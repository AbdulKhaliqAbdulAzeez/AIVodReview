## Tasks

### 1. Implement `generate_report()` in `pipeline/report.py`

**Function signature:**
```python
def generate_report(
    events: List[MistakeEvent],
    source_file: str,
    config: dict
) -> str:
    """
    Formats mistake events into a coaching report.
    Writes the report to disk and returns its file path.
    """
```

**Logic:**
1. If `events` is empty, generate a "no mistakes found" report (don't skip output entirely — a clean run should still produce a file).
2. Group events by `category` using `itertools.groupby` or a simple dict accumulator.
3. Sort groups alphabetically by category name for consistent output.
4. Within each group, sort events by `timestamp_ms` ascending.
5. Format each timestamp as `MM:SS` using `divmod(timestamp_ms // 1000, 60)`.
6. Compute summary statistics: total event count, most frequent category, average events per minute.
7. Write to file based on `config["output"]["report_format"]`:
   - `"markdown"` → `.md` file
   - `"json"` → `.json` file
   - `"both"` → write both

### 8. Write `SETUP.md` (user-facing setup guide)

Create a brief setup guide for anyone who wants to run the tool:

```markdown
# Setup Guide

## Requirements
- Python 3.11+
- NVIDIA GPU with CUDA 12.x drivers (recommended)
- Tesseract OCR 5.x (`sudo apt install tesseract-ocr`)

## Installation
1. Clone the repo.
2. Create and activate a virtual environment:
   python -m venv .venv && source .venv/bin/activate
3. Install Python dependencies:
   pip install -r requirements.txt
4. Verify Tesseract:
   python -c "import pytesseract; print(pytesseract.get_tesseract_version())"

## Calibration
Before processing footage, verify crop coordinates for your resolution:
   python training/calibrate_crops.py --frame path/to/screenshot.png

Adjust `ui_crops` values in `config.yaml` if needed.

## Usage
   python main.py --input path/to/your/vod.mp4

Output report will be written to `data/processed/`.
```

**Acceptance criteria:** A teammate can follow this guide from scratch and run the tool without asking questions.
