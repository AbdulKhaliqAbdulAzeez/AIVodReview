# Completed Sprints

No sprints have been completed yet. Completed sprint documents will be moved here from `upcoming/` or `active/` as they finish.

---

## How to Move a Sprint Here

1. When all tasks in a sprint's Definition of Done are checked off, move the file:
   ```bash
   mv sprints/active/sprint-XX-name.md sprints/completed/sprint-XX-name.md
   ```
2. Add a completion date and brief retrospective note at the top of the file before moving.
3. Move the next `upcoming/` sprint into `active/`.

## Retrospective Template

Add the following block to the top of any sprint file before moving it to `completed/`:

```markdown
## Retrospective
**Completed:** YYYY-MM-DD
**What went well:**
- 

**What was harder than expected:**
- 

**Carry-over or follow-up items:**
- 
```
